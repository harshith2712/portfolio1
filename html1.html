Projects

A Framework for Automated Generation of Floor Plan Images through Deep Learning.

This project focuses on automating the generation of 2D floor plans using deep learning. It employs LSTM networks to map user-provided text descriptions to architectural layouts. The system integrates a PyQt/PySide-based user interface for input collection, preprocesses textual data, and utilizes a trained model to generate accurate and functional floor plans. The approach aims to streamline the design process, making floor plan creation more accessible to a broader audience, including those with limited architectural knowledge.

ePharmaHub: A Web-Based Medicine Procurement System

The ePharmaHub: A Web-Based Medicine Procurement System is a Project in which sophisticated web-based platform designed to enhance accessibility and efficiency in pharmaceutical e-commerce. It enables users to seamlessly browse, order, and receive both prescription and over-the-counter medications from the convenience of their homes. The platform integrates advanced security measures, prescription validation, and a comprehensive product catalog to ensure a safe and user-friendly experience. Emphasizing regulatory compliance, data protection, and scalability, this solution sets a new standard for digital healthcare, offering a reliable and seamless online medication procurement system.
